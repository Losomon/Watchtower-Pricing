# Security Policy

## Supported Versions

| Version | Supported |
| ------- | -------- |
| 2.x     | ✅       |
| 1.x     | ❌       |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to security@watchtower.example.com

Do NOT report security vulnerabilities in public issues.

## What to Expect

- Acknowledgment within 48 hours
- Detailed timeline and remediation plan
- Credit in the security advisory (if desired)

##scope

This policy covers:
- Price scraping logic
- API authentication
- Data storage
- Alert notifications
