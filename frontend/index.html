Quick Start

1. Install
# Clone and install git clone https://github.com/you/watchtower-pricing cd watchtower-pricing pip install -r requirements.txt
2. Add a product
# Track a Jumia product, alert below KES 25,000 python automation/run.py add \ https://jumia.co.ke/sony-wh1000xm5/ \ --name "Sony WH-1000XM5" \ --target 25000
3. Run a check
python automation/run.py track # 🔭 Watchtower — checking 1 product # ✓ Sony WH-1000XM5: KES 28,500 (▼ 8.1%)
4. Start the API
uvicorn api.main:app --reload # → Swagger UI: http://localhost:8000/docs # → Health: http://localhost:8000/health # → Products: GET /products # → Track all: POST /track/all
5. Configure alerts
# .env ALERT_EMAIL_TO=you@gmail.com ALERT_EMAIL_FROM=alerts@gmail.com SMTP_PASSWORD=your-app-password TELEGRAM_TOKEN=your-bot-token TELEGRAM_CHAT_ID=your-chat-id
6. Run tests
pytest tests/ -v # tests/test_watchtower.py # ✓ TestProduct::test_valid_product # ✓ TestProduct::test_detect_store_from_url # ✓ TestPriceTracker::test_price_drop_detected # ✓ TestJsonRepository::test_price_history_ordering # ... 22 passed in 0.84s
